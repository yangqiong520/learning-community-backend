5. CORS 跨域问题解决

CORS 配置：
  • Origin: * (允许所有来源)
  • Methods: GET, OPTIONS
  • Headers: Content-Type, Authorization
  • 移除限制: 移除了 X-Frame-Options: SAMEORIGIN

解决方案：

  1. 确保后端服务器在运行
  2. 检查文件 ID 是否正确
  3. 前端和后端使用相同的协议（http 或 https）
  4. 如果仍有问题，检查浏览器控制台的具体错误信息
